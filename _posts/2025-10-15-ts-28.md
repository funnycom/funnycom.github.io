이제 AI의 도움을 최소화하고, 스스로 기능을 추가해봅시다.

### 💪 도전 과제

#### Level 1: 기본 고급 기능 (AI 힌트 허용)

**과제 1-1: 책 카테고리**
```
요구사항:
1. Book에 category 속성 추가
   - type Category = "fiction" | "non-fiction" | "science" | "history"
   
2. LibraryManager에 카테고리별 검색 추가
   - getBooksByCategory(category: Category): Book[]
   
3. 통계에 카테고리별 권수 추가

스스로 시도:
- Book 인터페이스 수정
- 생성자 수정
- 검색 메서드 구현

막히면 AI에게:
"Book에 선택적 속성 추가하는 방법만 알려줘"
```

**과제 1-2: 회원 등급 자동 승급**
```
요구사항:
1. Member에 borrowHistory: number 속성 추가
   - 총 대출 횟수 기록
   
2. 10권 이상 대출하면 VIP로 자동 승급
   - upgradeToVIP() 메서드 추가 (방법 A 선택 시)
   - 또는 새 VIPMember 객체 생성 (방법 B 선택 시)
   
3. borrowBook 시 자동 체크

스스로 구현:
- 어떻게 등급을 변경할까?
- 기존 대출 기록은 어떻게 유지할까?

힌트만:
"회원 객체를 교체하는 방법 알려줘"
```

**과제 1-3: 예약 시스템**
```
요구사항:
1. 대출 중인 책을 예약할 수 있음
2. 반납 시 예약자가 있으면 알림
3. 예약자 순서대로 처리

스스로 설계:
- Reservation 클래스 필요한가?
- Book에 reservationQueue 추가?
- 어떤 구조가 좋을까?

막히면:
"큐(Queue) 자료구조를 TypeScript로 구현하는 방법"
```

**과제 2-2: 도서 평점 시스템**
```
요구사항:
1. Book에 ratings: number[] 추가
2. addRating(memberId: string, rating: number) 메서드

4. 높은 평점 순 정렬

스스로 구현:
- 한 회원당 한 번만 평가 가능하게

- 정렬 알고리즘

힌트:
```

**과제 2-3: 대출 기록 통계**

```
스스로 구현:

- 정렬은?
```
