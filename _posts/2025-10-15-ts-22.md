## 📋 프로젝트 개요

### 무엇을 만들까요?

할 일(Todo)을 관리하는 웹 애플리케이션을 만들어봅시다. 사용자는 할 일을 추가하고, 완료 처리하고, 삭제할 수 있습니다. 또한 JSONPlaceholder API를 통해 서버와 데이터를 주고받는 기능도 구현합니다.

### 완성된 앱의 모습

```
=== Todo 앱 ===

📝 할 일 목록:
1. ⬜ TypeScript 공부하기
2. ✅ 프로젝트 설계하기
3. ⬜ API 연동하기

명령어:
- add [제목]: 할 일 추가
- toggle [번호]: 완료/미완료 전환
- delete [번호]: 삭제
- list: 목록 보기
- fetch: 서버에서 가져오기
- sync: 서버와 동기화
```

### 학습 목표

이 프로젝트를 완성하면:

- ✅ TypeScript로 실전 애플리케이션 설계 능력
- ✅ 인터페이스와 클래스 활용 능력
- ✅ REST API 연동 능력
- ✅ 에러 처리 능력
- ✅ AI와 협업하는 능력

### 예상 소요 시간

- **총 소요 시간**: 4-6시간
- **1일차 (2-3시간)**: 요구사항 분석, 설계, 기본 구조
- **2일차 (2-3시간)**: API 연동, 고급 기능, 테스트

### 난이도

⭐⭐⭐☆☆ (중급)

- TypeScript 기초 문법 필수
- async/await 이해 필요
- HTTP/API 기본 개념 필요

#### 💡 설계 예시

```typescript
// 1. 인터페이스
interface Todo {
  id: number;
  title: string;
  completed: boolean;
  userId: number;
}

type FilterType = "all" | "completed" | "active";

// 2. TodoManager 클래스
class TodoManager {
  private todos: Todo[];
  private nextId: number;
  private apiService: ApiService;
  
  constructor(apiService: ApiService) {
    this.todos = [];
    this.nextId = 1;
    this.apiService = apiService;
  }
  
  // 메서드들...
  addTodo(title: string): Todo
  deleteTodo(id: number): boolean
  toggleTodo(id: number): boolean
  getTodos(filter?: FilterType): Todo[]
  async fetchFromServer(): Promise<void>
  async syncToServer(): Promise<void>
}

// 3. ApiService 클래스
class ApiService {
  private baseUrl: string;
  
  async get<T>(endpoint: string): Promise<T>
  async post<T>(endpoint: string, data: any): Promise<T>
  async delete(endpoint: string): Promise<void>
}
```
